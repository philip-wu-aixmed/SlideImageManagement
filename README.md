### Naming Convention
#### Slide ID
The naming convention for Slide ID is broken down into following fields:
**XX**-**Y**-**ZZZZ**

**XX** :    medical institution;
        00: TUNGS
        01: VGHTPE

**Y** :     preparation type;
        0: Cytospin
        1: Thinprep
        2: Surepath

**ZZZZZ** : serial number
